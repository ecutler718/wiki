Arrays

An array is a datastructure used in programing languages for storing multiple pieces of data

Here are some examples of arrays:

This array contains strings:
["hello","Michael"]

This array contains integers:
[1,2,3,4,5]

This array contains a varible, integer, and a string:
[firstName, 1, "1"]

This array contains integers and an array:
[1, 2, [1, 2, 3, 4], 3, 4]
